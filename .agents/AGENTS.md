# Project-Scoped Rules

## Git Operations
- **Consent Required**: Never perform `git commit`, `git push`, or any other modifying Git write operations without obtaining explicit, prior consent from the user in the chat.
- **Descriptive Commit Messages**: All commit messages must be highly descriptive, explaining not just the high-level change, but also detailing the specific architectural updates, logic changes, and files modified. Avoid short or generic summaries.
